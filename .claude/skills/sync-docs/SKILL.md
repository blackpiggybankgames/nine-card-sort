---
## モデル分担
全手順を `Agent` ツール（`model=opus`）に委譲して実行する（調査専用スキルのため）。

## 目的
コードとドキュメントの乖離を検出する。

## 手順
1. CLAUDE.md を読む
2. docs/*.md の先頭20行だけ読む（全文は読まない）
3. コードのディレクトリ構造を確認
4. 主要ファイル（scenes/, scripts/, config/）の一覧を取得
5. 乖離の疑いがあるドキュメントを特定
6. 該当ドキュメントのみ詳細チェック
7. 差分一覧を人間に報告

## チェック観点
- ディレクトリ構造の記載が実態と一致するか
- 存在しないファイルへの参照がないか
- 新規追加ファイルがドキュメントに反映されているか

## 出力形式
```
## 乖離の疑いがあるドキュメント
- docs/XXX.md: 〜の構造が古い可能性
```

## 注意
- 修正は勝手にしない。報告のみ
- トークン消費が大きいため節目でのみ実行
- Task tool（サブエージェント）での実行を推奨
