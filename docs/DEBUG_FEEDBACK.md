## 書き方ガイド

### カテゴリ
- `[BUG]` - バグ（動作しない、エラーが出る）
- `[UI]` - UI/見た目の問題
- `[REQ]` - 要望・改善提案
- `[Q]` - 質問・確認事項

### 優先度
- `!!!` - 致命的（ゲームが動かない）
- `!!` - 重要（ゲーム進行に支障）
- `!` - 軽微（動作するが気になる）
- （なし） - 要望・提案

### 記入例

```
### [BUG] !!! カードが表示されない

**状況**: スタートボタンを押した後
**期待**: 9枚のカードが表示される
**実際**: 画面に何も表示されない
**再現**: 毎回発生する
**補足**: コンソールにエラーが出ている（スクショ添付）
```
