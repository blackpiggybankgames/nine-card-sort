## ゲームメカニクスの整理

### 山札の構造
```
位置0 [一番上] ← 手番で能力を使うカード
位置1
位置2
位置3
位置4 [真ん中] ← 9枚の中央（5番目）
位置5
位置6
位置7
位置8 [一番下]
```

### 手番の流れ
1. 位置0のカードの能力を「使う」or「使わない」
2. 能力を使う場合 → 効果を実行（発動カードは一時的に山札から除外）
3. 発動カード（位置0だったカード）を位置8（一番下）に移動

### 能力を使わない場合
- 位置0のカードがそのまま位置8へ移動
- 9回繰り返すと元の配置に戻る（1サイクル）

## 1と2だけでクリア可能か：数学的検証

### 結論：**可能**

### 証明

#### 基本操作の定義
- **能力なし移動**: 位置0 → 位置8（巡回シフト）
- **能力1（引き上げ）**: 任意のカードX → 位置0、その後カード1 → 位置8
- **能力2（押し下げ）**: 任意のカードY → 位置8、その後カード2 → 位置8

#### 任意の配置から目標配置への到達可能性

**主張**: 能力1と能力2の組み合わせで、任意の2枚のカードの相対順序を入れ替えられる。

**戦略例**: カードAをカードBの前に持ってきたい場合
1. カード1が位置0に来るまで待つ（最大8手番）
2. 能力1でカードAを位置0に引き上げる
3. カード2が位置0に来るまで待つ
4. 能力2でカードBより前にあるカードを押し下げる
5. 繰り返しで目標順序に近づける

**結論**: 置換群の理論により、能力1と能力2で生成される操作は対称群S9の任意の置換を生成可能。したがって任意の初期配置から任意の目標配置に到達できる。

#### 効率性

| 指標 | 評価 |
|------|------|
| 最悪ケース手番数 | 非常に多い（数十〜百手番以上の可能性） |
| 平均的な効率 | 低い（1枚を正しい位置に固定するのに複数サイクル必要） |

**基本方針との適合**: ✅ 「手番をかければクリア可能」を満たす

## 各能力の詳細解析

### 評価軸

| 軸 | 説明 |
|----|------|
| 自由度 | 選択肢の数（0=固定、高=任意選択可能） |

### カード1: 引き上げ

| 項目 | 値 |
|------|-----|
| 効果 | 任意の1枚を一番上へ移動 |
| 自由度 | ★★★★★ (8通りの選択肢) |
| 有用性 | ★★★★★ |
| 難易度 | ★☆☆☆☆ (低) |

**解析**:
- いつ使っても効果がある「万能能力」
- 「次に処理したいカード」を即座に手番に持ってこれる
- 失敗しにくく、初心者でも有効活用できる

**基本方針との適合**: ✅ 基盤能力として適切

### カード3: 中央配置

| 項目 | 値 |
|------|-----|
| 効果 | 任意の1枚を真ん中（5番目）へ移動 |
| 自由度 | ★★★★★ (8通りの選択肢) |
| 有用性 | ★★★★☆ |
| 難易度 | ★★☆☆☆ (低〜中) |
| 条件依存 | 低（カード5との連携で価値向上） |

**解析**:
- 自由度が1や2と同じく高い
- 単独でも「中盤の調整」に使える
- カード5との連携（3→5コンボ）で価値が上がる

**問題点**: ⚠️ **難易度が低すぎる**
- 1や2と同様に「任意の1枚」を対象にできる
- 基本方針「1→9で難易度上昇」に適合していない

### カード4: 2枚まとめ

| 項目 | 値 |
|------|-----|
| 効果 | 上から2枚をまとめて一番下へ移動（順序保持） |
| 自由度 | ☆☆☆☆☆ (固定操作) |
| 有用性 | ★★★☆☆ |
| 難易度 | ★★★☆☆ (中) |
| 条件依存 | 中（次のカードが適切である必要） |

**解析**:
- 発動時、カード4は除外されるため、実質「次のカード1枚」を一番下に送る
- 隣接する2枚の順序を保って移動できる点が特徴
- 「次のカード」を確認してから使うか判断が必要

**注意**: 発動カード除外ルールにより、実際の動作は設計意図と異なる可能性あり

### カード6: 3枚まとめ

| 項目 | 値 |
|------|-----|
| 効果 | 上から3枚をまとめて一番下へ移動（順序保持） |
| 自由度 | ☆☆☆☆☆ (固定操作) |
| 有用性 | ★★★☆☆ |

### カード8: 参照まとめ

| 項目 | 値 |
|------|-----|
| 効果 | 次のカードの数字X枚をまとめて一番下へ移動 |
| 自由度 | 間接的（次のカードに依存） |
| 有用性 | ★★★★☆（条件次第で非常に強力） |

- 予測が複雑で、熟練者向け

**基本方針との適合**: ✅ 高難易度・高リターンで適切

## シナジー分析

### 現在のコンボ

| コンボ | 手順 | 効果 |
|--------|------|------|
| 3→5 | 3で真ん中に配置 → 5で引き出し | 任意のカードを一番上へ（1と同等、2手番） |
| 2→7 | 2で一番下に配置 → 7で真ん中へ | 任意のカードを真ん中へ（3と同等、2手番） |
| 2→7→5 | 上記 + 5で引き出し | 任意のカードを一番上へ（1と同等、3手番） |
| 8フィニッシュ | [8, X, ...] の形で8発動 | X枚を一気に移動 |

### シナジーの評価

- 3→5コンボ: 2手番で能力1と同等 → 効率は劣るが、3と5が離れていても使える
- 2→7→5コンボ: 3手番で能力1と同等 → 効率悪いが、2,7,5の順番で来た時に有効

**観察**: 現在のシナジーは「基盤能力（1,2）の代替手段」が多く、「基盤能力より効率的」なシナジーが少ない
