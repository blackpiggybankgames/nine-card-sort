# ゲームフロー

## ゲームの進行順序

### 1. 準備フェーズ

ゲーム開始時、9枚のカードをシャッフルしてランダムな順番で山札を作る。
山札を全体が見えるように持つ（扇状に広げるなど）。

### 2. メインフェーズ

プレイヤーは以下の手番を繰り返して、目標の並び順を目指す。

#### 手番でできること（2択）

**選択肢A: 能力を使わずにカードを移動**
- アクティブカードをそのまま一番下に移動する

**選択肢B: 能力を使ってカードを移動**
- アクティブカードの能力を発動し、効果を実行する
- その後、アクティブカードを一番下に移動する

> **重要**: 能力を使っても使わなくても、アクティブカードは必ず一番下に移動する

### 3. 終了判定

山札が目標の並び順（1, 2, 3, 4, 5, 6, 7, 8, 9）になればクリア。
