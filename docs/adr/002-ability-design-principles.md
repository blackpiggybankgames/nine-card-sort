# ADR-002: カード能力の設計方針

## 状況

9枚のカードにそれぞれ能力を割り当てる際の設計指針が必要だった。

## 決定

1. **難易度曲線**: 手番をかければ誰でもクリア可能、少ない手番は難しい
2. **最低限の能力**: カード1と2の能力だけでクリア可能（ただし手番は多くかかる）
3. **能力の価値**: 3〜9は効率化のための能力、使いこなすと手番短縮
4. **難易度上昇**: 1→9にかけて使用タイミングの見極めが難しくなる

## 理由

- **クリア保証**: 1と2があれば数学的に任意の配置から目標に到達可能（置換群S9の理論）
- **段階的な学習**: プレイヤーが能力を覚えながら上達できる
- **戦略の深さ**: 効率的なクリアには3〜9の活用が必要
