# ADR-001: 発動カード除外ルール

## 状況

カード能力に「上からX枚を移動」等の記述がある。発動カード自身を対象に含めるかどうかを決める必要があった。

## 決定

能力発動前に発動カードを山札から除外し、残り8枚に対して能力を適用する。

手番の流れ:
1. 一番上のカード（位置0）を取り出す ← この時点で山札は8枚
2. 能力を発動（8枚に対して効果を適用）
3. 発動カードを一番下に追加 ← 山札が9枚に戻る

## 理由

- 「上から2枚」に自分自身を含めると、自分を2回移動する矛盾が生じる
- GAME_DESIGN.mdの「自分自身を移動させる記述は不要」という設計意図と一致
- 全能力で一貫したルールになる
