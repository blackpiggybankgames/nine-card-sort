# Game Design: ナインカードソート（Nine Card Sort）

## Core Concept
1〜9の数字が書かれた9枚のカードを、立ったまま手に持ちながら、各カードの能力を駆使して上から1・2・3…9の順番に並び替えるソリティアパズル。

---

## Game Rules

### 勝利条件
山札の一番上から順に 1, 2, 3, 4, 5, 6, 7, 8, 9 と並んだ状態になったらクリア。

### 敗北条件
なし（制限なし・何手でも試行可能）

### ゲームループ

ゲーム開始時、9枚のカードをシャッフルしてランダムな順番で山札を作る。
プレイヤーは以下の手番を繰り返して、目標の並び順を目指す。

1. 山札を全体が見えるように持つ（扇状に広げるなど）
2. 山札の一番上にあるカードの能力を「使う」か「使わない」かを選ぶ（任意）
3. 能力を使う場合 → カードに書かれた効果を実行する
4. 一番上のカードを一番下に移動させる（毎手番必ず実行）
5. 山札が目標の並び順になればクリア

### 重要なゲーム用語

| 用語 | 説明 |
|------|------|
| 山札 | 9枚のカードを全て表向きで重ねたもの |
| 一番上 | 山札の最上部にあるカード |
| 一番下 | 山札の最下部にあるカード |
| 真ん中 | 上から5番目のカード（9枚なので常に明確） |
| 次のカード | 現在の一番上のカードの、1つ下にあるカード |

---

## 重要なゲーム要素

### カード

- 枚数: 9枚（数字1〜9が1枚ずつ）
- 構成: 表面に「数字」と「能力テキスト」が記載
- 持ち方: 全て表向きで重ねて持つ（扇状に広げて全体を把握しながらプレイ）

### 手番でできること（2択）

**選択肢A: 能力を使わずにカードを移動**
- 一番上のカードをそのまま一番下に移動する

**選択肢B: 能力を使ってカードを移動**
- 一番上のカードの能力を発動し、効果を実行する
- その後、一番上のカードを一番下に移動する

> **重要**: 能力を使っても使わなくても、一番上のカードは必ず一番下に移動する

---

## 各カードの能力

能力の制約：
- 自分自身（一番上のカード）を移動させる記述は不要（手番処理で自動的に一番下に移動するため）
- 他のカードへの操作のみを記述する

| カード | 能力名（仮） | 効果 |
|-------|------------|------|
| 1 | 引き上げ | 任意の1枚を一番上へ移動する |
| 2 | 押し下げ | 任意の1枚を一番下へ移動する |
| 3 | 中央配置 | 任意の1枚を真ん中（5番目）へ移動する |
| 4 | 2枚まとめ | 上から2枚をまとめて一番下へ移動する（順序を保つ） |
| 5 | 中央引き出し | 真ん中（5番目）のカードを一番上へ移動する |
| 6 | 3枚まとめ | 上から3枚をまとめて一番下へ移動する（順序を保つ） |
| 7 | 底上げ | 一番下のカードを真ん中（5番目）へ移動する |
| 8 | 参照まとめ | 次のカードの数字をXとして、上からX枚をまとめて一番下へ移動する（順序を保つ） |
| 9 | 合計送り | 足して9になる2枚の組み合わせを一番下へ移動する（順序を保つ） |

### シナジーの例

- **3番 → 5番コンボ**: 3番で任意のカードを真ん中に置き、5番の能力でそれを一番上へ
- **2番 → 7番 → 5番コンボ**: 狙ったカードを2番で一番下に送り、7番で真ん中に上げ、5番で一番上に持ってくる
- **8番フィニッシュ**: 終盤に `[8, 4, 5, 6, 7, ...]` の形を作り、8番の能力で一気に整列させる

## 物理パラメータ（デジタル版）

※ 数値は仮の値。開発中に調整します。

- 解像度: 800 x 600
- カードサイズ: 100 x 150 units
- カードのアニメーション速度: 0.3秒
- 山札の扇状表示時の角度: 1枚あたり 8度
