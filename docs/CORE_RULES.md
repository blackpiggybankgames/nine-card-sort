# コアルール

## ゲームコンセプト

1〜9の数字が書かれた9枚のカードを、立ったまま手に持ちながら、各カードの能力を駆使して上から1・2・3…9の順番に並び替えるソリティアパズル。

## 基本ルール

### 勝利条件
山札の一番上から順に 1, 2, 3, 4, 5, 6, 7, 8, 9 と並んだ状態になったらクリア。

### 敗北条件
なし（制限なし・何手でも試行可能）

### カード
- 枚数: 9枚（数字1〜9が1枚ずつ）
- 構成: 表面に「数字」と「能力テキスト」が記載
- 持ち方: 全て表向きで重ねて持つ（扇状に広げて全体を把握しながらプレイ）

## カード番号の設計思想

カード番号1〜9は**難易度スケール**を表す。

- 番号が小さいほど（1に近いほど）：発動タイミングが単純、副作用が少ない、使いやすい
- 番号が大きいほど（9に近いほど）：発動タイミングが複雑、副作用が強い、使いにくい

この原則はすべての能力設計・カード割り当ての判断基準となる。
新しい能力を追加・変更する際は、この難易度スケールに整合しているかを確認すること。
